# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-22

### Added
- Right-click context menu for reading selected text with Explanatory and Storytelling voice modes
- Floating draggable control bar for playback controls and settings
- Live synced captions with focus mode highlighting
- Click-to-replay functionality for repeated playback
- Karaoke word marking for tracking read progress
- Local neural Piper TTS engine via WASM for offline synthesis
- Web Speech API fallback for broader browser compatibility
