Prompt caching

Reuse large, static parts of your prompt across requests to cut cost and latency.

Prompt caching lets you reuse the same context across many requests. Instead of resending a long prompt every time, Claude stores it and reads it back in a fraction of the time.

It is ideal for long system prompts, documents, and tool definitions that stay the same between calls. You pay less, and responses start faster.
