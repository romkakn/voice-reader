## שימוש
1. פותחים `demo-studio.html` בכרום (דאבל-קליק).
2. **📝 עריכת טקסט** — עורכים כותרות / פסקאות / פיצ'רים / CTA ישירות על המסך. שוב לחיצה לסיום.
3. **▶ נגן הדגמה** — תצוגה מקדימה (כולל קול נוירוני אמיתי דרך Web Speech).
4. **🎬 הקלט והורד וידאו** — בוחרים **“הכרטיסייה הזו”** + מסמנים **“שיתוף אודיו של הכרטיסייה”** → ההדגמה רצה ונשמרת כ-`voice-reader-demo.webm`.

> טיפ: להמרה ל-MP4 (פייסבוק/לינקדאין אוהבים mp4):
> `ffmpeg -i voice-reader-demo.webm -c:v libx264 -pix_fmt yuv420p -c:a aac voice-reader-demo.mp4`
